# Doctor-Appointment-App

This project is a full stack Doctor Appointment Booking System built using the MERN stack (MongoDB, Express.js, React, and Node.js). The application allows patients to browse doctors, view their specialties, and book appointments online. It also includes dashboards for doctors and administrators to manage appointments and system data.

The platform supports three user roles: patients, doctors, and admins. Patients can register, log in, search for doctors, and schedule appointments. Doctors can view and manage their appointments and update their profile information. Administrators can add doctors and manage the overall system through an admin dashboard.

The frontend was developed using React and Tailwind CSS to create a responsive and user-friendly interface. The backend was built with Node.js and Express, where RESTful APIs handle authentication, appointment booking, and data management. MongoDB is used to store user, doctor, and appointment data.
